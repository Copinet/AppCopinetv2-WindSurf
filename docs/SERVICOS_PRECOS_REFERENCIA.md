## 📊 IMPOSTO DE RENDA

**Declaração de Imposto de Renda (ÚNICO)**
- Preço PDF: R$ 60,00
- Preço PDF + Impressão: R$ 65,00
- Público: CLT, Aposentado, Pensionista
- NÃO fazemos: Ações, Criptomoedas, FIIs, BDRs, ETFs, Contratos Futuros, Renda Variável
- Fluxo: Fazemos pra Você (APENAS Lojas Copinet ou ADM)
- Prazo: 3 dias

### Opção 1: Com Senha Gov.br
**Dados necessários:**
- CPF
- Senha Gov.br (verificação 2 etapas DESATIVADA, nível Prata/Ouro)
- Nome
- Data Nascimento
- Endereço Completo
- Dados Bancários: Banco, Conta, Agência OU Chave PIX (se for CPF, dispensa dados bancários)

### Opção 2: Sem Senha Gov.br (Upload Documentos)
**Dados necessários:**
- CPF
- Nome
- Data Nascimento
- Endereço Completo
- Dados Bancários ou Chave PIX
- **Uploads:** Informe de Rendimentos (todas empresas ano anterior), RG, CPF

**Dependentes (Sim/Não):**
- Se SIM: Múltipla escolha
  - Filho(a): Menor 21 anos, não trabalhou ano anterior
  - Cônjuge: Não trabalhou ano anterior
  - Enteado(a): Menor 21 anos, não trabalhou ano anterior
  - Pai/Mãe: Verificar teto de rendimento
- Para cada dependente: Nome, CPF, Data Nascimento, Grau Parentesco
- Com senha gov.br: Dados dependentes NÃO necessários

**Campo:** Informações Complementares (textarea)

## 🖨️ IMPRESSÃO RÁPIDA (CARRO-CHEFE)

**Impressão de Documentos**
- Upload múltiplo: PDF, Word, JPG
- Conversão automática para PDF
- Contagem automática de folhas
- Preview antes do pagamento
- Propriedades por arquivo:
  - Tipo: P&B ou Colorido
  - Papel: Comum A4 / Cartão / Fotográfico
  - Cópias: Quantidade
  - Páginas específicas: Ex: 1-2, 5-10
- Preços:
  - P&B: R$ 1,00/folha
  - Colorida (comum): R$ 1,50/folha
  - Consultar tabela para outros papéis
- Mensagem: Campo texto para orientações
- Fluxo: Cliente Faz Sozinho

**Impressão de Fotos**
- Upload múltiplo: Imagens
- Tamanhos:
  - 10x15: R$ 3,00/foto
  - 13x18: R$ 5,00/foto
  - 15x21: R$ 7,00/foto
  - 21x29: R$ 9,00/foto
- Papel: Fotográfico
- Quantidade por foto
- Fluxo: Cliente Faz Sozinho

**Foto 3x4**
- Tirar selfie (múltiplas opções, escolher melhor)
- Edição automática: Fundo branco, centralizar, zoom
- 6 fotos em papel 10x15
- Preços:
  - Só JPG: R$ 5,00
  - Impressão folha inteira sem corte: R$ 8,00
  - JPG + Impressão cortada (6 fotos): R$ 14,00
